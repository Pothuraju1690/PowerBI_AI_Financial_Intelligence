# PowerBI AI Financial Intelligence Dashboard

## Project Overview

This project presents a comprehensive Financial Intelligence Dashboard built using Microsoft Power BI for analyzing Nifty 100 listed companies.

The dashboard transforms raw financial data into interactive business insights through KPI cards, visual reports, trend analysis, and financial performance monitoring.

---

## Objectives

- Analyze financial performance of companies.
- Evaluate profitability and health scores.
- Monitor cash flow and balance sheet strength.
- Assess debt and leverage risks.
- Track dividend and shareholder returns.
- Provide decision-making insights through visual analytics.

---

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Python (Data Preparation)
- CSV Datasets

---

## Dataset

The project uses cleaned financial datasets containing:

- Profit & Loss Data
- Balance Sheet Data
- Cash Flow Data
- Company Information
- Health Score Metrics

---

## Dashboards

### 1. Executive Overview
Provides high-level KPIs and company performance summary.

### 2. Profitability Analysis
Analyzes revenue, profit margin, and profitability indicators.

### 3. Financial Health Analysis
Evaluates company health scores and financial stability.

### 4. Cash Flow Analysis
Monitors operating activities, free cash flow, and net cash flow.

### 5. Balance Sheet Analysis
Analyzes assets, liabilities, equity, and borrowings.

### 6. Debt & Leverage Monitor
Tracks debt-to-equity ratio, leverage score, and interest expenses.

### 7. Dividend & Shareholder Returns
Evaluates dividend payout, EPS, and shareholder value.
